História Da Web

A web é um serviço criado pelo Tim Bernes Lee, em 1990. Que funciona sobre a internet, que é uma rede de computadores ,utilizando a arquitetura clientes - Servidor

Para isso ele definiu a estrutura de funcionamento da web,cliente servidor e criou a linguagem HTML,para exibição de documetos e o protocolo HTTP para comunicação entre a aplicação cliente e servidor.

Para cuidar dos padrões da web e propriedade da tecnologia ele criou a fundação w3c World Web Consorcium.

A referencia das tags HTML pode ser encontarada no site da w3c html reference
